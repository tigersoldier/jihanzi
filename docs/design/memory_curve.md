# 记忆曲线设计

记汉字使用 SM-2 间隔重复算法来安排汉字的复习，并根据学生对每个汉字的掌握程度动态调整复习间隔。

## 核心算法：SM-2 变体

### Ease Factor（难度系数）更新

每个汉字有一个 `ease` 值，起始值为 **2.5**，下限为 **1.3**。每次复习后根据评级按以下公式更新：

```
EF' = EF + ( 0.1 − (5 − q) × (0.08 + (5 − q) × 0.02) )
```

其中 `q` 为本次复习对应的分值（见下方评级映射）。

公式中的 **0.1、0.08、0.02** 是 SM-2 作者 Piotr Woźniak 根据记忆实验数据拟合出的固定常数，不可调整。

### 评级映射

| 等级 | 含义 | 对应 q | ease 变化 | 效果 |
|------|------|--------|-----------|------|
| a | 完全掌握，一次性书写正确 | 5 | EF + 0.1 | 间隔拉长 |
| b | 部分掌握，书写大体正确但有局部错误 | 3 | EF − 0.14 | 间隔小幅缩短 |
| c | 需要部分提醒才能想起如何书写 | 2 | EF − 0.32 | 间隔明显缩短 |
| d | 完全遗忘 | 0 | — | 重置 ease 为 2.5，间隔重置为 1 天 |

### 间隔计算

```
新间隔 = 当前间隔 × 新 ease
```

间隔计算结果**四舍五入**取整。

### d 级（完全遗忘）处理

- ease 重置为 2.5，间隔重置为 1 天
- 历史复习记录保留，供后续分析
- 当天以额外配额重新教学，不挤占新字配额
- 不算在"新学习的汉字"数量中

## 每日节奏

### 学习与复习交替

```
学新日 → 纯复习日 → 学新日 → 纯复习日 → ...
```

学新日后的下一天必定是纯复习日，严格交替。

### 每日配额

| 日类型 | 总字数上限 | 说明 |
|--------|-----------|------|
| 学新日 | 35 字 | 复习优先（上限 30），剩余配额填新字（5 个） |
| 纯复习日 | 30 字 | 全部为到期复习字 |

上限值可在系统中调整。

### 新字来源

从生字本按录入顺序依次挑选，每次 5 个。

## 同一天内重复巩固

### 规则

复习时评分为 c 或 d 的字，在当天全部复习完成一轮后，进入重复巩固环节：

- 只对 c/d 字进行额外轮次
- 最多重复 **3 轮**
- 超过 3 轮仍未掌握的计入统计，供后续分析
- SM-2 只使用**第一轮**的评分来更新 ease 和间隔——后续几轮是短期强化，不计入长期记忆模型

### 原因

SM-2 测量的是"间隔一段时间后的记忆保持"，不是"当天练了几遍后终于会了"。如果用重复练习后的评分喂入算法，会高估真实的记忆保持，导致间隔拉得过长，反而加速遗忘。

## 缺勤处理

如果某天未使用系统（孩子生病、外出等），恢复后：

- 学新/纯复习的交替节奏**不打断**——回来后按原日历继续
- 积压的到期复习字进入待处理队列
- 每日配额上限不变，队列中的字按优先级依次填充
- 孩子不因缺勤而欠"复习债"

## 新字初始值

新学汉字首次进入系统时：

- 起始 ease = 2.5
- 初始间隔 = 1 天（固定，不可调）
- 首次复习在学新日的次日（即下一个纯复习日）

## 参数汇总

| 参数 | 默认值 | 是否可调 |
|------|--------|----------|
| 算法 | SM-2 变体 | — |
| 初始间隔 | 1 天 | 否 |
| Ease 起始值 | 2.5 | 否 |
| Ease 下限 | 1.3 | 否 |
| 评级映射 | a=5, b=3, c=2, d=0 | 否 |
| 间隔取整 | 四舍五入 | 否 |
| 每日复习上限 | 30 字 | 是 |
| 每次新字数量 | 5 字 | 是 |
| 当天重复轮数上限 | 3 轮 | 是 |
