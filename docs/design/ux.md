# UX 设计

## 信息架构

### 底部导航栏（三入口）

| Tab | 用途 | 频率 |
|-----|------|------|
| 今天 | 今日复习/学新任务流，核心入口 | 每日使用 |
| 孩子 | 当前孩子学习数据面板，切换孩子 | 偶尔查看 |
| 生字本 | 生字本编辑器，增减字、调整顺序 | 偶尔编辑 |

设置入口放在顶部角落（齿轮图标），不占据底部导航。

## 首次使用

### 登录页

Google 登录为技术前置依赖（无登录无法访问 Drive），作为首个页面：

- 欢迎文案简述 app 功能
- "使用 Google 登录"按钮
- 纯前端 OAuth 流程

### 登录后

直接进入主界面。如未创建任何数据：

- 「今天」Tab 显示空状态引导："添加孩子和生字本开始学习"
- 「孩子」Tab 引导创建第一个孩子
- 「生字本」Tab 引导创建或导入生字本

不做线性引导流程。设置中常驻"导入数据"和"导出数据"按钮（详见设置页）。

## 今日任务页

### 结构

```
┌─────────────────────────────┐
│  📅 2026年6月28日  学新日     │
│                             │
│  ●●●●○○○○○○○○  12 / 35      │ ← 进度条 + 数字
│                             │
│  ┌─────────────────────────┐ │
│  │                         │ │
│  │          花              │ │ ← 楷体大字（参考写法）
│  │                         │ │
│  │  huā  花朵·花园·花草     │ │ ← 拼音 · 组词
│  │                         │ │
│  └─────────────────────────┘ │
│                             │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐   ← 电脑端四按钮横排
│  │  ✓   │ │  ~   │ │  ?   │ │  ✗   │
│  │ 完全 │ │ 部分 │ │ 需提 │ │ 遗忘 │     手机端改为两行两列
│  │ 掌握 │ │ 正确 │ │ 示   │ │      │
│  └──────┘ └──────┘ └──────┘ └──────┘
└─────────────────────────────┘
```

### 任务流

1. 顶部显示当天类型（学新日 / 纯复习日）
2. 进度条显示当前轮次进度（已完成 / 总字数）
3. 任务流以列表呈现：复习到期字排前，新字排后（仅学新日有新字）
4. 卡片信息：楷体大字 + 拼音 + 组词
5. 家长叫孩子纸上书写 → 对照 → 按评级按钮

### 评级按钮

**电脑端**：四按钮水平排列 `[完全掌握] [部分正确] [需提示] [遗忘]`

**手机端**：四按钮两行两列

| a·完全掌握 | b·部分正确 |
| c·需提示 | d·遗忘 |

颜色梯度：a 绿色 → b 黄色 → c 橙色 → d 红色

### 评级确认动画

- 按下评级后，字上闪过对应颜色的勾（0.3 秒）
- 确认过后下一个字滑入

## 当天重复巩固

### 轮次衔接

第一轮全部完成后，弹出确认卡片：

> "X 个字需要再巩固，是否开始第 2 轮？"

- [跳过，今天就到这]
- [开始巩固]

如第 2 轮后仍有 c/d 字，同样确认进入第 3 轮。最多 3 轮。

### 重复轮 UX

- 卡片样式与第一轮相同
- 左上角增加轮次标记：「第 2 轮」「第 3 轮」
- 超过 3 轮仍未掌握的字记入统计，可在「孩子」Tab 查看

## 完成庆祝页

全部任务完成后，显示庆祝页：

```
       ✓
  今天完成 26 个字
    完全掌握  18
    部分正确   5
    需提示     2
    遗忘       1

    [返回首页]
```

- 不在庆祝页列出 c/d 字的具体内容
- c/d 字的追踪在「孩子」Tab 的统计中查看

## 生字本 Tab

### 生字本编辑器

```
┌─────────────────────────────┐
│  生字本：人教版一年级上册  ▾  │ ← 切换生字本
│                             │
│  ┌─────────────────────┐    │
│  │ 添加新字 > 雨_______ │    │ ← 快速添加输入框
│  └─────────────────────┘    │
│                             │
│  共 124 字                  │
│                             │
│  1.  一                     │
│  2.  二                     │
│  3.  三                     │
│  4.  上                     │
│  ...                        │
│  125. 雨  ← 新增            │
└─────────────────────────────┘
```

- 所有字按学习顺序纵向排列
- 顶部快速添加：输入汉字 → 回车添加
- 支持拖拽调整顺序（电脑端）/ 长按拖动（手机端）
- 支持删除（滑动或右键删除）

## 空状态

各 Tab 在无数据时的引导：

**今天（无孩子或无生字本）**：
> "还没有学习记录。去「孩子」添加学习者，去「生字本」准备字库吧。"

**孩子（无孩子）**：
> "添加第一个孩子开始追踪学习进度。"

**生字本（无生字本）**：
> "创建生字本或从文件导入，为学习准备字库。"

## 响应式行为

| 场景 | 电脑 | 手机 |
|------|------|------|
| 评级按钮 | 四按钮水平排列 | 四按钮两行两列 |
| 生字本拖拽排序 | 鼠标拖拽 | 长按拖动 |
| 今日任务流 | 卡片居中，最大宽度限制 | 全宽卡片 |
| 设置页 | 单栏居中 | 全宽，滚动查看 |
