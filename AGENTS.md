## Agent 技能

### Issue 跟踪器

GitHub Issues；外部 PR 与 issue 一同作为分流入口。详见 `docs/agents/issue-tracker.md`。

### 分流标签

全部五个标准角色使用其默认标签字符串：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。详见 `docs/agents/triage-labels.md`。

### 领域文档

单上下文布局——仓库根目录下一个 `CONTEXT.md` 和 `docs/adr/`。详见 `docs/agents/domain.md`。
